## 🛠 Installation

### Step 1: Run Installer

Launch `install.bat` **as Administrator** to synchronize all key dependencies and integrate Python 3.12.0, creating the perfect foundation for growth.

### Step 2: Start the Server

After successful installation, run `start.bat` to activate our server.

When you see this iconic interface, you're on the right track:

